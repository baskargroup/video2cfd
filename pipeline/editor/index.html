Controls

  • Left click — Select object
  • Right drag — Orbit camera
  • Middle drag / Scroll — Pan / Zoom
  • W — Translate mode
  • E — Rotate mode
  • Delete / Backspace — Delete selected object
  • Ctrl+Z — Undo last action
  • Ctrl+C — Copy selected object
  • Ctrl+V — Paste copy (offset to the side, same Z & rotation)
  • Ctrl+D — Duplicate selected (copy + paste in one step)
  • Ctrl+S — Save edits
  • Escape — Deselect / Close help
  • G — Toggle Z=0 grid
  • ? — Toggle this help
  • + Add button — Add STL from template library (mannequins, furniture, etc.)
  • Room panel — Preview and rotate the room enclosure box (Step 5)

Workflow

1. Click objects to select them. Use the Move/Rotate gizmos or the properties panel to adjust.

2. Delete false positives with the Delete key or the sidebar button.

3. Use + Add to place additional STL templates (mannequins, furniture). They are auto-scaled to match the scene.

3b. Use + Procedural Table to add a new geometry-generated table. Select it to edit its dimensions and click Regenerate to update the mesh. For existing procedural tables from Step 4, select them to see and edit their dimensions inline.

4. Click Save to write edited placements, or Save & Rebuild to also regenerate the combined STLs.

Click anywhere or press ? / Esc to close.
